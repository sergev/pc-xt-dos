Intel PLM Compiler 3.1 for DOS
Released in 1987 by Intel
For DOS

PLM is a high-level programming language with many low-level
features, originally developed in 1972 by Gary Kildall. PL/M was the
first high-level language for microprocessor-based computers, and was
used to implement CP/M.

The Intel PLM Compiler is a tool that compiles PL/M to 8088/8086 
machine code.
